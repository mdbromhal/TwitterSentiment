# TwitterSentiment
ABSTRACT: Using Sentiment Analysis to Determine Airline Sentiment on Twitter

- Cleaning the data
- Tokenizing the words
- Simple RNN
- Train on and predict data
- Label encoding:
    - 0 = neutral
    - 1 = positive
    - 2 = negative

Ex) tweet = 'This airline is so fun' --> Predicted class:  [1] (correct!)
